## How to Run the Seed

### Prerequisites
1. Database is set up and running
2. Prisma migrations are applied: `npm run prisma:push`

### Run the Seed
```bash
cd backend
npm run prisma:seed
```

Or directly with tsx:
```bash
npx tsx prisma/seed.ts
```
